Garden Sensor

Matter over Thread · XIAO ESP32-C6

What this installs

Air temperature Air humidity Air pressure Soil temperature Soil moisture Battery level

Before you flash

  1. Connect your XIAO ESP32-C6 via USB-C
  2. Click Install below
  3. Select the correct COM / serial port
  4. Wait for flashing to complete (~30 s)
  5. Open Apple Home → Add Accessory → More Options
  6. Scan the QR code or enter the pairing code shown on the serial monitor
Chrome or Edge required. ESP Web Tools uses the Web Serial API, which is not supported in Safari or Firefox.
